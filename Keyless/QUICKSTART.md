## Common Issues & Quick Fixes

### Issue: "Keycaps folder not found"
**Fix:** Make sure you're in the right game and the game has keycaps to collect.

### Issue: UI doesn't appear
**Fix:** 
1. Check if script executed without errors
2. Try restarting Roblox
3. Use a different executor

### Issue: Teleport not working
**Fix:**
1. Click "Scan Keycaps" first
2. Wait for scan to complete
3. Then click "Start Teleport"
